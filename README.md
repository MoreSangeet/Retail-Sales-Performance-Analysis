# Retail-Sales-Performance-Analysis
# 🛒 Retail Sales Performance Analysis

## 📌 Project Overview

This project analyzes retail sales data using Python to identify sales trends, top-performing products, seasonal demand, and business insights. The analysis follows a complete data analysis workflow including data cleaning, feature engineering, exploratory data analysis (EDA), and data visualization.

The project demonstrates practical use of Python libraries to convert raw sales data into meaningful business insights that can support data-driven decision-making.

---

# 🎯 Project Objectives

- Analyze retail sales performance.
- Identify the highest revenue-generating product categories.
- Find the top-performing products.
- Analyze monthly and quarterly sales trends.
- Calculate profit and profit margins.
- Generate business recommendations based on data insights.

---

# 📂 Dataset Information

The dataset contains retail transaction records with the following features:

- Order ID
- Product Name
- Category
- Quantity
- Unit Price
- Revenue
- Date
- Region

### Dataset Summary

- Original Records: **10,557**
- Records after Data Cleaning: **10,494**
- Missing Revenue Values Filled: **47**
- Duplicate Records Removed

# 📈 Key Findings

- Electronics generated the highest total revenue.
- Laptop Pro was the highest revenue-generating product.
- Revenue increased significantly during Quarter 4.
- December recorded the highest monthly revenue.
- January recorded the lowest monthly revenue.
- Electronics contributed approximately **63.6%** of the company's total revenue.

---

# 💡 Business Insights

- Electronics is the company's primary revenue-generating category.
- Quarter 4 shows strong seasonal demand.
- High-performing products should receive priority in inventory planning.
- Promotional campaigns during January could improve sales during the slowest month.

---

# ✅ Business Recommendations

- Increase Electronics inventory before Quarter 4.
- Focus marketing campaigns on top-selling products.
- Offer promotional discounts during January.
- Improve sales strategies for lower-performing product categories.
